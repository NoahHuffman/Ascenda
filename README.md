Ascenda: A Student Schedule Organizer

Before you begin, ensure you have the following installed:

- Node.js (version >=20.11.1) & NPM

1. **Clone the repository**

   ```bash
   git clone https://github.com/NoahHuffman/Ascenda.git
   cd ascenda

   ```

2. **Install dependencies**
   Note - you must do this everytime you pull from the repository!!

   ```bash
   npm install

   ```

3. **Set up environment variables**
   SKIP THIS UNTIL WE HAVE API KEYS! Copy the .env.example file to .env.local and fill in the necessary API keys and database credentials.

   ```bash
   cp .env.example .env.local

   ```

4. **Run the development server**
   ```bash
   npm run dev
   Open http://localhost:5173 with your browser to see the result.
   ```
